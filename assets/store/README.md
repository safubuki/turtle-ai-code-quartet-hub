# Store 画像素材チェック

Microsoft Store 申請前に、ここへ公開用スクリーンショットや追加画像を置いて確認します。
個人のワークスペース名、ユーザー名、パス、APIキー、未公開コード、社内情報が分からない画像だけを使ってください。

## 必須

- デスクトップ用スクリーンショットを最低1枚用意する。
- 推奨は4枚以上。
- PNG形式にする。
- 1366 x 768 px 以上にする。
- 1ファイル50 MB以下にする。

## 推奨ファイル名

- `desktop-01-main.png`: 4面表示の全体像（A-D スロットに異なるアプリを起動した状態）
- `desktop-02-compact.png`: 縮小モード（常時最前面の操作バー）
- `desktop-03-focus.png`: スロット選択と1面フォーカス表示
- `desktop-04-ide-cli.png`: スロットの IDE / CLI 切替（VS Code と AI CLI の選択）

## 撮影前チェック

- サンプル用の空フォルダやダミーリポジトリを開く。
- 4面のうち少なくとも1面は AI CLI（Codex / Copilot / Gemini / Grok / Claude）を表示し、マルチアプリ対応が伝わる構図にする。
- VS Code やターミナルのエディタ、履歴、最近開いた項目に個人情報がないことを確認する。
- Store掲載文案と矛盾する機能を出さない。
- Microsoft、Visual Studio Code、GitHub、OpenAI、Anthropic、Google、xAI の公式アプリであるように見える表現を避ける。
